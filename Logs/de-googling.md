# de-googling log
## SMARTPHONE
**2024 -** i purchased a google pixel phone, which has the most long-standing support from Lineage OS. i will boot Lineage OS onto the phone in a couple of years. i am not rushing to do this as my banking apps are not yet supported and they are pretty useful! 

**2026 -** i currently use a bunch of apps from fdroid. see: https://keepandroidopen.org/
## EMAIL 
**2026 -** i am moving away from primarily using gmail. i have created different email accounts for different purposes, hosted on protonmail and tutamail. i also have a custom domain with purelymail (cheap).  later down the line, i will move my custom email domain onto [disroot](https://disroot.org/). I'll retain the gmail account throughout the move. this is a long and tedious administrative task. 
## WEB DOMAIN
**2023 -** got my web domain with porkbun to host my simple static website exclusively in html and css. 
## VPN
**2025 -** proton vpn is good because it allows port forwarding and they regularly have sales. hosted in EU so subject to more stringent privacy laws, and has a no-logs policy.
## WEB HOSTING
**2023 -** move from krystal to github pages. ideally will set up my own server to host my little static website when i am less itinerant. 
## PHOTOS
**2026 -** i'm very very slowly moving away from google photos towards ente.io. right now, i'm just auto backing everything up on ente.
## BROWSER
i have been using brave for a long time now after testing out a lot of alternatives to chrome. brave is good because it is chromium based. it runs like chrome.
